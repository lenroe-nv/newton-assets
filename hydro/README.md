# Hydroelastic Simulation Assets

## Overview

This package contains custom-modeled USDA assets used for hydroelastic contact simulation in Newton.

| Asset | File | Description |
|-------|------|-------------|
| Cup | `cup.usda` | A simple cup mesh |
| Gripper Pad | `pad.usda` | A pad mesh for gripper fingers |
